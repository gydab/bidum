# Gydabjörg Portfolio

Einföld persónuleg heimasíða (MVP) með eftirfarandi köflum:

- Forsíða (hero)
- Um mig
- Portfolio / verkefni
- Samband

## Keyrsla

Þetta er static síða og þarf enga uppsetningu.

1. Opnaðu möppuna í VS Code.
2. Opnaðu `index.html` í browser (eða með Live Server ef þú vilt).

## Sérsníða efni

- Breyttu texta í `index.html`.
- Bættu við eða breyttu verkefnum í `script.js` (`projects` fylkið).
- Stílaðu frekar í `styles.css`.

## Næstu skref

- Tengja raunveruleg verkefni (GitHub/live links).
- Setja inn prófílmynd.
- Bæta við sér síðu eða kafla fyrir ferilskrá (CV) ef óskað er.
